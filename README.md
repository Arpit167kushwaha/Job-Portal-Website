# Job Portal Website

A full-stack Job Portal application developed using **Java, Spring Boot, React.js, MySQL, Spring Security, and JWT Authentication**. The platform enables recruiters to post jobs and candidates to search and apply for jobs through a secure and user-friendly interface.

---

## Features

### Authentication & Security

* JWT-based Authentication
* Role-Based Authorization
* Secure Login & Registration
* Spring Security Integration

### Candidate Features

* Create and Manage Profile
* Search Available Jobs
* Apply for Jobs
* Upload Resume
* Track Application Status

### Recruiter Features

* Post New Jobs
* Update Job Listings
* View Applicants
* Manage Posted Jobs

### Admin Features

* Manage Users
* Monitor Job Listings
* System Administration

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/job-portal.git
```

### Backend Setup

```bash
cd backend
mvn spring-boot:run
```

### Frontend Setup

```bash
cd frontend
npm install
npm start
```
